=============== START THE PROGRAM ==============

- locate to the www directory located within the source folder
- run the command: php -S localhost:<port_number>

!!! We recommend using PHP 8.1.9 and above.

!!! During the testing process, we observed that the program works best using MacOS and 
WSL (Window subsystem for Linux), please use those operating system for the best result.
